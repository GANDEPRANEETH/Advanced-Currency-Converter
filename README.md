# 💱 Advanced Currency Converter (Flask)

A full-stack web application built using Flask that converts currencies in real-time using an exchange rate API and stores conversion history in a SQLite database.

---

## 🚀 Features

- Real-time currency conversion
- Uses Exchange Rate API
- Stores conversion history in SQLite
- Error handling for invalid inputs
- Clean UI with HTML & CSS
- Flask backend architecture

---

## 🛠 Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS
- REST API (Exchange Rate API)
